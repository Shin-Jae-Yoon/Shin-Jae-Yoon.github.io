# 01. 불 자료형이란?


불 자료형은 참(True)과 거짓(False)를 다루는 자료형이다.

  • True : 참

  • False : 거짓


단순히 참과 거짓을 판별하는 것 이외에도 자료형을 참과 거짓으로 판단할 수 있다.

  • 문자열, 리스트, 튜플, 딕셔너리, 집합 등의 값이 비어있으면 거짓, 있으면 참

  • 0은 False, 1은 True

  • None은 False

파이썬은 모든 것을 객체로 다룬다. 그 중 NoneType 클래스의 유일한 객체인 None은 값 자체가 없거나 존재하지 않거나 등의 경우를 의미한다. None에 대한 자세한 내용은 개발 블로그에 추가로 포스팅했다.
